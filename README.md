# shortstat

**`git log --shortstat` summary.**

## Quickstart

```diff
λ shortstat .

+  200,534  insertions
-  131,906  deletions
~    6,750  files changed
```

## Installation

```sh
npm install --global shortstat
```

## Options

* **`--author`**: _string_, the git author
* **`--branch`**: _string_, the git branch

## Development

* **`yarn fmt`**: formats the code
* **`yarn build`**: builds the code
